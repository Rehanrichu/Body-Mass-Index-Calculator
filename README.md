# BMI (Body Mass Index) Calculator 
### Description
This is a health-tech tool that calculates a person's Body Mass Index (BMI). It uses a person's weight and height to determine if they are in a healthy range, which is a key topic in our Biology studies.
### What it does
 It allows the user to enter precise numbers like 1.75 meters using float.
 
It calculates BMI by dividing weight by the square of the height ($BMI = kg/m^2$).

Depending on the result, it gives a specific status (Underweight, Healthy, Overweight, or Obese). 

### How to use it
Run the script.

Enter your weight in kilograms and height in meters.

The AI will tell you your BMI and give you a health suggestion.
